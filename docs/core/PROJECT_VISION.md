# Project Vision

Velvet Secret je platforma za simbolni dialog.

Uporabnik lahko pride z vprašanjem, sanjami, tarot branjem ali občutkom.
Sistem mu ne pove, kaj se bo zgodilo.
Sistem mu pomaga videti možne pomene.

Velvet Secret povezuje:
- sanje,
- tarot,
- Jungovo psihologijo,
- simbolni atlas,
- kabalo,
- numerologijo,
- gematrijo,
- mitologijo,
- filozofijo zavesti,
- refleksijska vprašanja.

Osrednja ideja:
Simbol ni napoved.
Simbol je vrata v uvid.

Dolgoročni cilj:
zgraditi lasten simbolni operacijski sistem, kjer AI pomaga človeku opaziti vzorce, povezave in notranje pomene, vendar nikoli ne prevzame njegove svobodne presoje.
