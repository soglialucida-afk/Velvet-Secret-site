# Architecture

## Plasti sistema

```text
User
↓
Emotion Layer
↓
Context Layer
↓
Knowledge Layer
↓
Reflection Engine
↓
Response Layer
↓
Journal / Memory Layer
```

## User Layer
Uporabnik vnese vprašanje, sanje, tarot vprašanje ali simbol.

## Emotion Layer
Sistem zazna ali vpraša po občutku.

## Context Layer
Sistem upošteva področje:
- odnosi,
- delo,
- družina,
- denar,
- osebni razvoj,
- duhovnost,
- ne vem.

## Knowledge Layer
Sistem išče po modulih:
- Jung Core,
- Dream Dictionary,
- Symbol Atlas,
- Tarot,
- Kabbalah,
- Mythology,
- Consciousness,
- Reflection.

## Reflection Engine
Sistem sintetizira pomen.
Ne poda ene resnice.
Poda več smeri.

## Response Layer
Odgovor je jasen, miren in varen.

## Journal Layer
Uporabnik lahko shrani sanje, simbole, vprašanja in uvide.
