# Journal Page

## Namen

Dnevnik omogoča uporabniku spremljanje sanj, kart, simbolov in uvidov.

## Funkcije

- shrani sanje,
- shrani tarot branje,
- označi simbole,
- spremljaj ponavljajoče motive,
- mesečni povzetek,
- vprašanja za refleksijo.

## Pravilo

Dnevnik ne napoveduje.
Dnevnik opazuje vzorce.
