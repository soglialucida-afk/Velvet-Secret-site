# Slike za stran Branje sanj

Mapa za vse slike, ki se uporabljajo na `sanje.html` in sorodnih straneh.

## Predlagana struktura

- `hero-*.{jpg,webp}` — hero sekcija
- `symbol-*.{jpg,webp}` — ilustracije za slovar simbolov
- `blog-*.{jpg,webp}` — slike za blog kartice
- `pricing-*.{jpg,webp}` — slike za cenik kartice

## Format
Priporoča se WebP za produkcijo (manjše datoteke), PNG/JPG za izvirnike.
